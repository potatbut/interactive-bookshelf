# 📚 Моя полка

Уютная виртуальная библиотека, где можно собрать свой книжный стеллаж, расставить книги, добавить декор и оформить маленькую комнату вокруг него ✨

Проект сделан на чистом JavaScript и Canvas. Это pet-проект для rofls: без сложной регистрации, без тяжелого бекенда и без лишней серьезности — просто красивая интерактивная полка для себя и друзей.

## 🌷 Что можно делать

- собирать 2D-стеллаж и менять его цвет, размер и количество полок;
- добавлять книги и настраивать их цвет, высоту, ширину и название;
- ставить книги корешком или обложкой;
- выбирать значки для книг через OpenMoji;
- искать эмодзи для книг, декора и вида из окна;
- перетаскивать книги и предметы по полкам;
- добавлять уютный декор: растения, лампы, свечи, игрушки и другие OpenMoji-иконки;
- настраивать комнату: стену, окно, коврик, кресло и вечерний свет;
- смотреть все книги отдельным списком по алфавиту;
- удалять и редактировать предметы прямо со сцены.


## 👤 Профили

В проекте есть простые локальные профили: пользователь вводит имя и пароль, а библиотека сохраняется отдельно для этого имени.

Сейчас все хранится только в браузере на текущем устройстве. Это удобно для прототипа, но не заменяет настоящую регистрацию и синхронизацию между устройствами.

## 🎨 Иконки

В проекте используются [OpenMoji](https://openmoji.org/) — открытые SVG-эмодзи с милым, простым и очень подходящим для такого проекта стилем.
