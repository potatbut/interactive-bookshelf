# Моя полка

Интерактивный прототип виртуальной библиотеки на чистом JavaScript и Canvas.

## Запуск

Откройте `index.html` напрямую или запустите любой статический сервер, например:

```powershell
npx serve .
```

## Что уже работает

- настройка размеров, цвета и числа полок;
- создание книг с размером, цветом и жанровой иконкой;
- название на корешке и обложке, размер шрифта и живой предпросмотр;
- положение книги корешком или обложкой;
- декор, ковры, окно, кресло и вечернее освещение;
- настройка формы, узора, цвета и размера ковра, окна и кресла;
- изменение названия библиотеки с синхронизацией заголовка вкладки;
- сохранение активного пользователя после обновления страницы;
- локальные цветные SVG OpenMoji для декора и изменение их масштаба;
- фильтр оттенка, предпросмотр и редактирование выбранного декора;
- автоматическая выкладка на пол предметов, не помещающихся по высоте или ширине;
- четыре локальных варианта вида из окна с текстовым поиском;
- свободное перемещение окна по стене, а кресла и коврика по полу;
- восемь видов кресел из пользовательского SVG, масштабирование и отражение;
- подтверждения перед удалением и сбросом, подсказки для кнопок;
- перетаскивание книги под курсором и автоматическая выкладка слишком высоких книг на пол;
- меню книги по двойному клику с удалением и живым редактированием;
- локальные профили и отдельное сохранение библиотеки каждого пользователя.

## Важное про авторизацию

Сейчас профили работают через `localStorage`: это удобно для прототипа, но данные не синхронизируются между устройствами. Пароль сохраняется только в виде SHA-256-хеша, однако настоящей серверной защитой это всё равно не является. Для публикации лучше подключить Supabase и хранить там серверный хеш пароля и JSON состояния комнаты.

Минимальная серверная модель:

- `profiles`: `id`, `username`, `password_hash`, `created_at`;
- `libraries`: `id`, `profile_id`, `name`, `scene_json`, `updated_at`.

Canvas-состояние уже собрано в одном объекте `state`, поэтому `scene_json` можно сохранять целиком без изменения формата редактора.

## Декоративные иконки

Используются локальные SVG из проекта [OpenMoji](https://openmoji.org/), распространяемые по лицензии CC BY-SA 4.0. Копии выбранных файлов находятся в `assets/openmoji`.

Варианты кресел подготовлены из переданного пользователем файла `16369163_Different colorful chairs and armchairs vector illustration set.svg` и находятся в `assets/chairs`.
